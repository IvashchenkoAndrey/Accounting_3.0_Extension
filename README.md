Расширение для типовой конфигурации Бухгалтерия предприятия, редакция 3.0 (3.0.173.37)

1. Доработана форма документа Поступление (акты, накладные, УПД).
   - В документе Поступление (акты, накладные, УПД) для вида операции Товары в шапку формы документа выведены ставки НДС, которые есть в табличной части документа.
      Информацию о ставках НДС размещена в правой части шапки формы в виде гиперссылки. Ставки НДС отсортированы по возрастанию.
2. Добавлен реквизит Ответственный (Тип реквизита — справочник Физические лица) в типовой справочник Статьи движения денежных средств.
3. Добавлен в конфигурацию регистр Оплаты по статьям ДДС.
4. Реализованы движения типовых документов по регистру Оплаты по статьям ДДС.
   - Документы Списание с расчётного счёта и Выдача наличных с видом операции Оплата поставщику делают движения по регистру Оплаты по статьям ДДС.
